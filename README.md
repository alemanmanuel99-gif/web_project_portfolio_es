# Portafolio

## Nombre del Proyecto

Portafolio Personal de Desarrollador

## Descripción del Proyecto y Funcionalidad

Es un sitio web personal que presenta tu perfil profesional como desarrollador web. El portafolio incluye:

Sección de presentación personal con foto y descripción
Galería de proyectos realizados
Información de contacto y habilidades técnicas
Diseño completamente responsivo que se adapta a móviles, tablets y escritorio
Navegación intuitiva y experiencia de usuario optimizada
Tecnologías y Técnicas Utilizadas
HTML5 semántico para estructura del contenido
CSS3 avanzado con técnicas de diseño responsivo
Unidades relativas (rem, %, vw/vh) y función calc()
Media queries para adaptación a diferentes dispositivos
Flexbox y Grid para layouts modernos
Trabajo con Figma para interpretar especificaciones de diseño profesionales
Herramientas de depuración (Device Toolbar de Chrome)
Técnicas de IA para acelerar el desarrollo y comprensión del código
